A beadandó feladat célja egy több modulból álló webalkalmazás elkészítése volt, amely különböző JavaScript technológiák gyakorlati alkalmazását mutatja be. Az alkalmazás egy mozi adatbázis köré épül, amely filmek adatait kezeli.

A fejlesztés során törekedtünk arra, hogy minden feladatpont külön modulban, jól elkülönítve, ugyanakkor egységes megjelenéssel valósuljon meg.
